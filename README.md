# Crystal Game

Each crystal has been given a random (hidden!) value. Click the crystals to increase your counter. Try to make your counter match the target number! If you go over, you lose :(

## Technologies Used
* Bootstrap
* jQuery
* Docker/Caprover (Deployment)
